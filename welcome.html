BranchKit Browser is ready

Navigate any website without a mouse — keyboard first, voice optional.

Start here

On any web page, press F. A short badge appears on every link, button, and field. Type the badge's letters to click it — that's the whole idea.

  1. Press F → badges appear
  2. Type a badge (e.g. as) → it activates
  3. A Capital opens it in a new tab; Esc exits

Optional — hands-free voice. Install the BranchKit desktop app and the same badges become speakable: say a badge's codeword to click, type, or navigate. No keyboard needed. The extension works fully without it.

Works on any site, with no account. Nothing leaves your browser — page data is used only to draw badges, and only reaches the optional desktop app on your own machine.
