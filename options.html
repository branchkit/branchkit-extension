Keyboard shortcuts

Rebind the in-page keyboard commands. Each command can have more than one key. While hints are visible, bare letters type hint codewords, so a key tagged hints hidden only fires with hints off — use Shift+letter or a Ctrl/Alt/Cmd chord (tagged always) to work in both. Shortcuts sync across browsers.
